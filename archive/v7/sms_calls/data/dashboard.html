About TTGO T-Call Project

    

      
Features

      
    
        
  • SMS sending and receiving with auto-forwarding
    • 
        
  • Incoming call management with auto-reply
    • 
        
  • NTFY push notifications for real-time alerts
    • 
        
  • Gate ESP integration for access control
    • 
        
  • Admin number management with special privileges
    • 
        
  • Persistent configuration via SPIFFS
    • 
        
  • Web-based dashboard for easy configuration
    • 
        
  • Remote settings management
    • 
      

      
      
Configuration Persistence

      
All settings are automatically saved to SPIFFS storage and persist across device reboots. 
         Changes made through this dashboard take effect immediately and are permanently stored.

      
      
API Endpoints

      
    
        
  • GET /api/config - Get current configuration
    • 
        
  • POST /api/config - Update configuration
    • 
        
  • POST /api/send_sms - Send SMS
    • 
        
  • POST /api/call - Initiate call
    • 
        
  • POST /api/hangup - Hang up call
    • 
        
  • GET /api/messages - List stored messages
    • 
        
  • POST /api/messages/delete_all - Delete all messages
    • 
        
  • POST /api/test_forward - Test forwarding connection
    • 
        
  • POST /api/check_settings - Check for remote settings updates
