# v8 Logging Standard

## Tags
- BOOT
- MODEM
- WIFI
- CONFIG
- FIREBASE
- RATE_LIMIT
- SMS
- CALL
- API

## Rules
- One meaningful line per event.
- No noisy repetition.
- Errors must explain what failed and what happens next.

## Acceptance Criteria
- Logs are readable at a glance.
- Startup summary is always printed.
- Debug output is structured and professional.
